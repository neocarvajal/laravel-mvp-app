### MVP CRUD LARAVEL APP 

## Start App

- clone repo
- cd (repo dir)
- composer install (in console)
- set .env file with user and password for database
- php artisan serve (in console)

## Database

- import database (puntodeventa.sql)
- php artisan migrate:fresh --seed ((in console))
